# Win

Ctrl + X	删除

Ctrl + C	复制

Ctrl + V	粘贴

Ctrl + Z	撤销

Ctrl + D	复制当前行

Shift + Enter	向下另起一行

Ctrl + Alt + Enter	向上添加一行

Shift + Left/Right/Home/End	向左/向右/行头/行尾选择

Ctrl + Shift + up/down	选择行向上下移动

Ctrl + Shift + F12	切换全屏代码
